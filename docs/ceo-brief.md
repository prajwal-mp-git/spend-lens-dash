CEO Briefing: Spendlens Expense Dashboard

What I built and why it helps us?

I built a new dashboard to track our company expenses. Before this, the finance team was wasting a lot of time checking exchange rates on Google to make their monthly reports.

This app does all that work automatically. It takes all our expenses from different countries, changes them into US Dollars, and puts them in a clear table. I also added a Search Bar to instantly find specific expenses (like "AWS") and an "Export to CSV" button so the finance team can download the table directly into Excel. Finally, there is a slider where you can change the Euro rate and instantly see how it changes our total budget. This will save our team hours of boring manual work and stop math mistakes.

Three things I skipped for now and why?

1. No online database: I chose to store the data locally on the web page instead of setting up a cloud database. I did this so I could finish a working visual app for you to see today, rather than spending all my time on the backend setup.
2. Fixed exchange rates: I used a fixed list of exchange rates instead of live ones from the internet. Sometimes live rate websites go down or charge money, so using fixed rates made sure the app would be 100% stable for this demo.
3. Basic form checking: The form blocks people from leaving fields blank, but it doesn't do advanced checks like looking for duplicate receipts. I wanted to make sure the main math worked perfectly first before spending time on complex rules.

What I want to build next?

If I had another week to work on this, I would focus on:
1. Saving the data online: This is the most important. I would connect a database so the whole team can log in and add expenses together without losing their work.
2. Connect to Accounting Software: Now that we have CSV exports, the next step would be connecting the app directly to software like QuickBooks to send the data over automatically.
3. Uploading photos: I would add a feature so employees can upload a picture of their physical receipt when they add an expense.
